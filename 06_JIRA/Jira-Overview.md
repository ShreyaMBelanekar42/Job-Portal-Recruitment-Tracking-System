This project was managed using Jira following Agile Scrum methodology.

- Created Epics for major modules: 
    1. User Authentication & RBAC
    2. Job Application, Resume Upload & Storage
    3. Auto-Shortlisting, Candidate Management System
    4. Interview Scheduling
    5. Interview Feedback Module
    6. Notifications System, Dashboards & Reports
    7. Admin & System Configurations, 8. Integrations (Email/Google Meet)
- Broke Epics into User Stories with acceptance criteria
- Further divided stories into technical and functional subtasks
- Planned and executed sprints
- Tracked progress using Scrum board and sprint reports
